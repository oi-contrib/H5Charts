series-map

地图主要用于地理区域数据的可视化，配合custom个性化等可以绘制更丰富的图表。

所有属性

series-map. type = 'map'
string

series-map. map = ''
string

使用 registerMap 注册的地图名称。

series-map. data
Array

地图系列中的数据内容数组。

series-map.data. name
string

数据所对应的地图区域的名称，例如 '江苏'，'南京'。

series-map.data. value
number

该区域的数据值。

series-map. label
Object

图形上的文本标签，可用于说明图形的一些数据信息，比如值，名称等。

series-map.label. show
boolean

是否显示标签。
