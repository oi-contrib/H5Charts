series-pie

饼图主要用于表现不同类目的数据在总和中的占比。每个的弧度表示数据数量的比例。

所有属性

series-pie. type = 'pie'
string

series-pie. startAngle = -90
number

起始角度。

series-pie. angle = 360
number

跨越角度，支持范围 (0, 360] 。

series-pie. radius = []
Array Function

表示饼图的内外半径，可以为如下类型：

  • [number,number]：分别表示内半径和外半径。
  • Function：函数参数为({width,height})=>Array，返回值格式需要是上面的数组。

series-pie. center = []
Array Function

表示饼图的圆心位置，可以为如下类型：

  • [number,number]：分别表示中心横坐标和纵坐标。
  • Function：函数参数为({width,height})=>Array，返回值格式需要是上面的数组。

series-pie. data
Array

饼图系列中的数据内容数组。

series-pie.data. name
string

数据项名称。

series-pie.data. value
number

数据值。
