series-custom

自定义系列可以个性化的绘制任意图表。

所有属性

series-custom. type = 'custom'
string

series-custom. draw
Function

可以使用画笔绘制任意需要的内容： 

draw:function(params){
    // 个性化绘制
}

上面的形参params包括：

  • painter 画笔 （具体可用方法可以查阅 oi-contrib.github.io/VISLite ）。
  • grid 包含画布尺寸信息（left、right、top、bottom、width、height）。
  • cx 画布中心横坐标。
  • cy 画布中心纵坐标。
