visualMap-continuous

连续型视觉映射组件（visualMapContinuous）。

所有属性

visualMap-continuous. min
number

指定 visualMapContinuous 组件的允许的最小值。 'min' 必须用户指定。

visualMap-continuous. max
number

指定 visualMapContinuous 组件的允许的最大值。 'max' 必须用户指定。

visualMap-continuous. inRange
Object

定义在选中范围中的视觉元素，可选的视觉元素有：

color 

visualMap:{
    ...,
    inRange: {
        color: ['lightskyblue', 'yellow', 'orangered']
    }
}
