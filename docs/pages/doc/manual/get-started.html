快速上手

获取 H5Charts

H5Charts 提供了多种安装方式，你可以根据项目的实际情况选择以下任意一种方式安装。

从 npm 获取 

    npm install --save h5charts

从 CDN 获取

可以从以下免费 CDN 中获取和引用 H5Charts：

  • jsDelivr

引入 H5Charts

如果是CDN的方式，使用Script标签引入后即可使用：

我们更推荐使用npm的方式： 

    import H5Charts from "h5charts";

绘制一个简单的图表

在绘图前我们需要为 H5Charts 准备一个定义了高宽的 DOM 容器： 

<body>
    <!-- 为 H5Charts 准备一个定义了宽高的 DOM -->
    <div id="root" style="width: 600px;height:400px;"></div>
</body>

然后，我们就可以创建图表了： 

var mychart = new H5Charts(document.getElementById('root'),{
     
    // 图表的配置项和数据
    title: {
        text: "H5Charts 入门例子"
    },
    grid: {
        top: 70
    },
    color: ["red"],
    xAxis: {
        data: ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
    },
    series: [{
        data: [120, 190, 150, 80, 70, 110, 130],
        type: "bar"
    }]
});

这样你的第一个图表就诞生了！

如果后续需要对图表进行调整（包括数据改变），可以使用mychart上的方法，比如： 

mychart.setOption({
    color: ["pink"]
});

现在，柱状图就变成粉红色了。
